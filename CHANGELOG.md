# Changelog

## v0.9

Production Candidate initial release.

### Added

- Domain-independent Project OS
- Implementation Task / Project Core Review Task separation
- Unit Review Agent pattern
- Direct Task Collaboration Protocol
- Project Core Gate lifecycle
- REWORK → Auto Fix → Resubmit loop
- PASS → Next Work Package event
- Long-running runtime guidance
- Project customization templates
- Root-relative artifact delivery standard
- Capability proof documents
- Optional Platform Adoption Status and Project Maturity metadata in the Project Context template
- Optional Current Phase and Phase Roadmap metadata in the Phase Kickoff template

### Known Limitations

- Platform-specific Task APIs may change
- Long-running availability limits depend on execution platform
- Multi-reviewer and multi-implementation orchestration are not standardized
- Notification and external monitoring are outside this release
