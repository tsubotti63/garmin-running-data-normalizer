# Application Evidence

This directory tracks public or explicitly public-safe evidence that may
support a future Codex for Open Source application readiness review.

It contains evidence indexes and trackers, not raw Garmin exports, generated
personal output, private review packs, application authorization, or a
substitute for current product requirements and release notes.

## Current tracker

- [`codex_application_evidence_tracker_v0_1.md`](codex_application_evidence_tracker_v0_1.md)

New evidence must identify a verifiable source, validation status, privacy
classification, limitations, and the application claim it could support.
Human approval remains required before application material is finalized or
submitted.
