# Documentation Language Convention v0.9

## Principle

- Concepts / Identifiers / Status / Paths: English
- Human explanations / background / review rationale: Japanese
- Machine-readable values: stable English tokens
- Chat output to Human: Japanese by default

## Standard Style

```text
HUMAN_DECISION_REQUIRED

既存ルールでは判断できないため、Humanの価値判断が必要。
```

## Canonical Terms

- Implementation Task
- Unit Review Agent
- Project Core Review Task
- Project Core Gate
- Review Package
- Evidence
- PASS
- REWORK
- HUMAN_DECISION_REQUIRED
- AUTO_FIX
- CONTINUE

同義語を無制限に増やしません。
