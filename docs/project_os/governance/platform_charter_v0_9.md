# Platform Charter v0.9

**Status**: Production Candidate  
**Authority**: Platform Governance

---

## 1. Purpose

HumanとAIが長期Projectを安全かつ自律的に推進するための最上位原則を定義します。

---

## 2. Principles

### Human Owns Values

Humanは目的・優先順位・許容リスク・正式採用を所有します。

### AI Owns Routine Judgment

既存ルールで判定可能な実装・QA・修正・Evidence生成・レビューをAIへ委譲します。

### Evidence First

自信ではなくEvidenceで判断します。

### Autonomous within Boundaries

定義済みBoundary内では自律実行します。

### Report, Don't Stop

改善は通常業務です。安全な改善は実装・検証・報告して継続します。

### AI Reviews AI

Implementationの自己評価だけで重要Gateを通過しません。

### PASS Means Continue

PASSは終了ではなくNext Work Package Start Eventです。

### Human Escalation Is Exceptional

Humanは既存ルールで解決できない判断だけを担当します。

---

## 3. Non-goals

本Platformは以下を目的としません。

- AIへ無制限の権限を与える
- Humanを完全に排除する
- すべてのProjectへ同一のDomain Ruleを強制する
- Evidenceなしで自律性を優先する

---

## 4. Project Customization

各Projectは、次を独自に定義してください。

- Project Purpose
- Domain Principles
- Project Boundary
- Definition of Done
- Data / Source / Legal / Security Rules
- Formal Close Authority
