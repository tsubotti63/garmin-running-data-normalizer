## Responsibility Boundary

- AI Collaboration Architecture defines collaboration principles and boundaries.
- Project OS defines standard project structure and conventions.
- Project Factory initializes projects.
- Project Governance Extension governs continuous operation, maintenance, review, and feedback.
- Platform Evolution evaluates reusable improvements for ACP standard adoption.
