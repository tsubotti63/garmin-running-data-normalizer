# Improvement Policy v0.9

## Levels

### Auto Apply

機能減少なし・Contract維持・Rollback可能なら、自律実装して報告。

### Candidate

複数案・Scope外・標準化未判断なら、Improvement Candidateとして継続。

### Human Decision

価値・Scope・不可逆変更を伴う場合のみ停止。

## Improvement Record

- ID
- Observation
- Evidence
- Change / Proposal
- Impact
- Applied
- Review Result
- Platform Promotion Candidate
- Status

## Principle

改善の基本動作は、停止ではなくEvidence付き実装・報告です。
