# v0.1.0-rc.2

## Release summary

This candidate expands the local-first workflow from the RC1 activities-only
Golden Path to a bounded multi-family Run-All command and a reproducible
analysis handoff. It remains a prerelease and is not published to PyPI.

## Included

- Activities-required Run-All with optional Gear, Personal Records, and bounded
  FIT sessions/laps
- Fixed deterministic output layout, QA, manifest, completion summary, and
  explicit partial-success behavior
- Bounded large-archive compatibility with existing safety controls preserved
- Private public-safe real-export execution evidence
- Analysis Handoff, reusable prompt, public synthetic usage example, three
  key-free analysis examples, and a Primary Case Study

## Reproducibility and privacy

The private validation completed with status `PASS`, exit code 0, unchanged
input, and two independent byte-identical outputs. The privacy check passed.
No private rows, dates, filenames, paths, identifiers, counts, or fingerprints
are published.

Real exports and generated output remain local. The Activities CSV stable key
may incorporate a source activity ID and must be removed from any externally
shared derivative.

## Known limitations

- Activities are required and optional families use exact filename rules.
- FIT support covers selected session/lap fields, not complete FIT behavior.
- Weather, Sleep, HRV, Parquet, dashboards, notebooks, automatic Analysis Pack
  generation, stable API guarantees, stable release, and PyPI are absent.
- Analysis examples are descriptive and do not provide medical or coaching
  conclusions.
