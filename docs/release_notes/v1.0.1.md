# Garmin Running Data Normalizer v1.0.1

Status: prepared release note; `v1.0.1` is not yet tagged or published.

`v1.0.1` is a packaging-only patch candidate for the initial package-index
publication of Garmin Running Data Normalizer. It preserves the `v1.0.0`
runtime behavior, public CLI, Run-All output contract, dependency boundary,
and supported Garmin dataset scope.

## Packaging and publication

- Advances the Python distribution version to `1.0.1`.
- Includes the PyPI-safe README and repeatable wheel/source-distribution gate.
- Includes the manual-only, build-only-by-default publication workflow.
- Requires exact source and version validation, target-specific approval,
  protected GitHub Environments, and Trusted Publishing through OIDC before
  either index upload.
- Verifies wheel and source-distribution installation before upload and checks
  the exact installed version after publication.

## Compatibility

This patch does not change the stable `1.x` CLI or versioned Run-All output
contract. It does not add runtime dependencies, hosted processing, Garmin
authentication, or new data families to Run-All.

## Security and privacy

Only synthetic fixtures are public. Real Garmin exports and generated personal
results remain local and must not be uploaded as package, Release, or CI
artifacts.

## Publication boundary

Preparing this file does not authorize or perform a Git tag, GitHub Release,
Trusted Publisher or Environment configuration, TestPyPI upload, or PyPI
upload. Those operations require explicit Human approval and recorded
post-publication verification.
