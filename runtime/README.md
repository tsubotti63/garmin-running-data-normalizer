# Runtime

AI Taskを作成・起動・連携するための実行資産です。

- `work/task_creation/`: 新規Task作成時に貼り付けるPrompt
- `work/implementation/`: Implementation Task Runtime
- `work/review/`: Project Core Review Runtime
- `agents/`: Sub-Agent Prompt
- `packages/`: Review Package標準

Project固有ルールはRuntime Promptを直接改変するより、Project Runtime Addendumとして追加してください。
