# Garmin Running Data Normalizer Runtime Addendum

## Mandatory reading

- `docs/project/phases/phase0_1/reading_order.md`
- Current Work Package and relevant `docs/reference/` evidence

## Autonomous actions

Local Target implementation, synthetic tests, deterministic QA, scans, review
evidence, and boundary-local rework.

## Stop conditions

Rights/license/publication decisions, privacy-boundary change, external resource
creation, irreversible operation, or scope/source-of-truth change.

## Project Core gates

Review Packages go directly to the Target Project Core Review role. Its `PASS`
is the sole Project Core Gate for implementation. Prior extraction reviews are
historical context, not active approval dependencies.

## Output requirements

Human chat is Japanese and concise. Evidence records changed files, tests,
negative paths, scans, manifest/hash, Unit Review, blockers, and next action.
